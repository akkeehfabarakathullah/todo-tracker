# Todo App- backend

A fast and lightweight backend API for managing todos using Node.js, Express, and MongoDB.

---

## Core Features

* Get all todos
* Create a todo
* Update a todo
* Delete a todo
* Toggle done / not done
* REST API structure
* MongoDB database integration

---

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

---

## Getting Started

## Installation

```bash
npm install
```

## Run Backend

```bash
cd backend
node server.js
```

Backend runs at:
http://localhost:5000

---

## API Endpoints

GET /api/todos
POST /api/todos
PUT /api/todos/:id
PATCH /api/todos/:id/done
DELETE /api/todos/:id
---

## Notes

* Start backend before frontend
* Make sure MongoDB is connected
* CORS should be enabled
